About MathWar

MathWar turns graphing, arithmetic, and problem solving into quick rounds that are easy to replay. The goal is simple: make practice feel active enough that players want one more try.

  1. What it is

    MathWar is an open-source collection of math mini-games for the browser. Each game focuses on a specific skill and keeps the session short enough for practice, warmups, or friendly competition.

  2. Suggestions

    Have an idea for a new mini-game, mode, or practice challenge? Open a feature request.

  3. Open project

    Bugs, ideas, and balance notes are tracked on GitHub. Visit the issue tracker or browse the repository.

  4. Contributing

    Want to help improve MathWar? Read the contributing guide before opening a pull request.
